The Course

Object Oriented Pattern Design

ICT 2207 is a core Java course in the Department of Information & Communication Technology at MBSTU, taught by Prof. Dr. Ziaur Rahman. It covers the foundations of object-oriented programming — classes and objects, encapsulation, inheritance, polymorphism, and abstraction — and how these ideas combine into recognizable design patterns.

Alongside the coursework, I'm running a self-paced Java practice plan to reinforce every topic with real code. Each module below links straight to the matching folder in my practice repository.
